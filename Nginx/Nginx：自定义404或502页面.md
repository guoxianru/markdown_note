# Nginx：自定义404或502页面

## 导读

> 当服务器出错时，访问域名会报404或502错误，如果想展示信息给用户的话，必须要自定义界面。

```shell
nginx配置：
server {
# 配置nginx502错误配置
error_page 502 /502.html;
location = /502.html {
root /usr/share/nginx/html;}
# 配置nginx404错误配置
error_page 404 /404.html;
location = /404.html {
root /usr/share/nginx/html;}
}
```

 配置完再去 /usr/share/nginx/html 目录下添加自定义的 404.html 和 502.html 文件即可。
